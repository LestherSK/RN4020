# RN4020
Arduino library for the RN4020 BLE module. The library is based off the following [datsheet](http://ww1.microchip.com/downloads/en/DeviceDoc/70005191B.pdf).

# Connection
|MCU             |BLE            |
| :--------------|:---------------|
| UART TX        | UART RX        |
| UART RX        | UART TX        |
| VDD 3.3V       | VDD 3.3V       |
| GND            | GND            |
